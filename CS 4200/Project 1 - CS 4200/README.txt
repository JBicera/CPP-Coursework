CS 4200 Project 1: 8-Puzzle A*Search Solver
by Joshua Bicera

How to Run
1.Navigate to folder from your terminal
2.Run python file using command "python project1_4200.py"

Interaction
Upon starting the program you will be presented with a menu
You can interact with the program using keyboard inputs in the terminal
[1]Single Test Puzzle: Test either a randomly generated puzzle or user inputted puzzle
[2]File Input: Test a file of puzzle boards by inputting the file name
[3]Exit: Exits the program to terminal
