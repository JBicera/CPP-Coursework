8 - Queen Problem Solver
Project 2 by Joshua Bicera

This Python program is designed to solve the 8-Queen problem using both Steepest Ascent Hill CLimbing
and Genetic Algorithm. The goal of this program is to find a configuration of a chess board where 8
queens are able to be placed on the board without being able to attack one another.

How to Run:
To run the program simply execute the command in your terminal where the code is located

python project2_CS4200.py
